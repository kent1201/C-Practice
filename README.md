## 🚀 如何執行

1. 確認已安裝 **C++14** 或以上版本的編譯器（如 `g++`, `clang++`）。
2. 切換到指定章節資料夾，例如：

   ```bash
   cd ch03
   g++ exercise3_1.cpp -o exercise3_1
   ./exercise3_1
   ```
3. 不同檔案為對應練習題目，可單獨編譯與執行。

## 📖 學習目標

* 依章節循序漸進，練習 C++ 的核心語法與觀念
* 熟悉 **STL (Standard Template Library)** 的常用容器與演算法
* 建立良好的 **程式架構與模組化設計**
* 為進一步學習 **演算法設計與系統開發** 打下基礎
